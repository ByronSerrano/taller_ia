## Requisitos
- Python 3.* 
- pip (instalado con Python por defecto)

## Instalación
1. Clona el repositorio en tu máquina:

   ```bash
   git clone https://github.com/ByronSerrano/taller_ia
   ```

2. Instalar dependecias:

   ```bash
   pip install -r requirements.txt
   ```
